## Prerequisites

* Linux
* PHP >= 8.2
* Xdebug
* Make
* Git
* Docker
* Laravel 12

## Setup

```bash
make compose-setup
```

## Start

```bash
make compose-up
```

## Run Linter and Tests

```bash
make compose-lint
make compose-test
```
